## D10 (Dasamsa) — 事业深度盘

**D10 Lagna = Libra**

事业基调：**Libra=Venus 管**——意味着你事业领域的"调性"会偏向 Venus 主题：**关系、平衡、美感、沟通、和谐、咨询、协作、艺术、教育、品牌**。这与 D1 的 AmK=Venus 信号**完全一致**——双盘共振确认事业方向。

### D10 关键发现

| 配置 | 含义 |
|------|------|
| **D10 Lagna = Libra** | 事业风格=Venus主题：关系/美/平衡/咨询 |
| **D10 第9宫 (Gemini) 行星聚集 = Sun + Jupiter + Venus** | 三吉星+Sun聚9宫 → 事业的"灵感来源/福分/老师"维度极强 |
| **D10 10宫(Cancer) = Moon 入庙** | 事业核心由 Moon 主导 → "情感连接/沟通/服务/照顾"是事业的实际操作模式 |
| **D10 4宫(Capricorn) = Mars(R) + Mercury** | 事业的"基础设施"由 Mars+Mercury 提供 → 行动力+智慧/沟通是底层 |
| **D10 11宫(Leo) = Saturn** | 事业收益由 Saturn 管理 → 长期积累型收益，非短期暴利 |
| **D10 2宫(Scorpio) = Rahu** | 收入渠道带 Rahu 色彩 → 跨界/新兴/非传统/有放大效应 |
| **D10 8宫(Taurus) = Ketu** | 事业突变/转折点有 Ketu 色彩 → 关键时刻的"放下/重启"会出现 |

### D10 核心结论（给 vedic-career 的预热）

1. **事业方向调色板**：Venus 主题（关系/美/咨询/教育/服务/品牌）+ Mercury 主题（沟通/分析/信息/智慧）+ Moon 主题（情感连接/服务/照顾）三重叠加。**不是单一方向，是"复合型"事业**。
2. **行业可能性**（仅指方向，不定具体职业）：
   - 教育/咨询/培训/治愈类（Venus+Moon+Jupiter 三吉聚 D10 9宫）
   - 文化/品牌/创意/设计（Venus+Mercury+D10 Libra Lagna）
   - 跨界/新媒体/平台经济（Rahu in D10 2宫=收益渠道）
   - 心理/服务/沟通型咨询（Moon 入庙 D10 10宫）
3. **事业曲线**：Saturn 管 D10 11宫收益 = **慢炖型积累，30 岁后才进入复利期**。30 岁前不要追求"快速变现"，那不是你的盘的玩法。
4. **关键时间窗口**：
   - **Ju-Ve AD (2026-2028)**：事业方向重大启动/转向期（Venus 同时管D10 Lagna+1宫主自家）
   - **Saturn MD (2034-2053)**：事业稳定积累期，Saturn 在 D10 11宫=收益慢但稳
   - **Mercury MD (2053+)**：事业终极兑现期（NBRY 释放）

## D4 (Chaturthamsa) — 财产/不动产盘

**D4 Lagna = Pisces**

物质舒适度基调：**Pisces=Jupiter 管**——意味着你的"物质安宁"主题由 Jupiter（大运主+AK）的状态决定。Jupiter D1 自家强 → D4 Lagna 由强 Jupiter 管 = **物质基础整体有保障**。

### D4 关键发现

| 配置 | 含义 |
|------|------|
| **D4 Lagna=Pisces** | 物质基础由 Jupiter 主导，整体有保障 |
| **D4 4宫(Gemini) = Moon + Mercury** | 不动产/家宅核心由 Mercury+Moon 管 → 房子可能跟"工作/迁移/沟通节点"绑定 |
| **D4 2宫(Aries) = Sun + Rahu** | 资产积累带 Sun 入旺+Rahu 放大 → 财产增长方式带"爆发+扩张"色彩 |
| **D4 11宫(Capricorn) = Mars(R) + Md** | 收益领域有 Mars 逆行+Maandi(凶星增强) → 收益过程有反复，需谨慎 |
| **D4 7宫(Virgo) = Jupiter** | 伴侣/合作领域有 Jupiter 加持 → 婚姻可能带来"物质升级" |
| **D4 8宫(Libra) = Ketu** | 突变/继承领域 Ketu → 意外财/继承可能性低或精神化 |

### D4 核心结论

1. **不动产/房产**：Pisces Lagna + Jupiter 强 = 有不动产潜力，但 Mars 在 D4 11宫逆行 = **首套房可能经历反复或晚买**
2. **财产积累方式**：Sun+Rahu 在 D4 2宫 = **跨界扩张+权威加持型积累**，而非保守储蓄
3. **婚姻与财产**：D4 7宫 Jupiter = 伴侣领域跟你的物质上升相关（不是"靠对方"，是"伴侣加成"）
4. **Ketu in D4 8宫 + Mars in D4 11宫**：**意外财（继承/遗产/中奖）不是你的盘的玩法**，老实劳动收益+长期积累是主线

## D5 (Panchamsa) — 权力/影响力盘

**D5 Lagna = Pisces**

权威/创造力基调：与 D4 同样 Pisces Lagna，意味着你的"影响力/权力气场"也由 Jupiter 主导。这是个好信号——你的影响力倾向于**"智者型/教育型/道德型"**而非强权/技术冷感型。

### D5 关键发现

| 配置 | 含义 |
|------|------|
| **D5 Lagna=Pisces** | 影响力风格=Jupiter主题：智者、教育者、有道德号召力 |
| **D5 1宫(Pisces) = Venus** | Lagna 同宫有 Venus = 个人魅力+美感+亲和力是核心影响力载体 |
| **D5 11宫(Capricorn) = Jupiter** | Jupiter 在 D5 11宫 = **通过社群/收益领域释放影响力**（不是单打独斗型） |
| **D5 12宫(Aquarius) = Sun + Mars(R)** | 影响力背后有 Sun+Mars 的"暗中能量" → 你的影响力带"为他人服务/损耗自我"的色彩 |
| **D5 10宫(Sagittarius) = Saturn** | 事业领域 Saturn = 严肃/责任型权威 |
| **D5 9宫(Scorpio) = Rahu+Ketu同宫** | 信仰/老师领域有节点同宫 → 信仰/价值观经历过深度重塑 |
| **D5 7宫(Virgo) = Mercury + HL** | 伴侣/合作领域 Mercury 入旺 → 合作伙伴质量高 |

### D5 核心结论

1. **影响力来源**：不是权力/职位，而是**"个人魅力+智慧+道德感+教育能力"**的组合（Venus+Jupiter主导）
2. **创造力释放**：Mercury 入旺在 D5 7宫 = **你的创造力通过"合作/对话/反馈"释放，不是孤独闭门型**
3. **权威气场出现时间**：Jupiter MD 内逐步建立，Saturn MD（35-54岁）固化为社会权威
4. **D5 Rahu-Ketu 同宫在 9宫 Scorpio**（D5 特殊规则，对宫错位）：信仰/价值观体系经过深度重组——这跟你 5宫 Ketu 的"反复审视"信号一致
